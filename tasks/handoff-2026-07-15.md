## Stage 24 固定动作

```text
explicit --representation snapshot-json
  -> explicit project-relative --envelope
  -> path allowlist + size/UTF-8/duplicate-key/schema/secret validation
  -> envelope content id + scope id
  -> fixed handoff producer with validated path
  -> Stage 18 reference consumer
  -> handoff source/id validation
  -> fixed snapshot producer with the same validated path
  -> snapshot source/id/canonical hash validation
  -> envelope content-drift recheck
  -> bounded scoped JSON result
```

实际调用：

```bash
python tools/codex_desktop_snapshot_json_reader.py \
  --project-root . \
  --representation snapshot-json \
  --envelope adapters/execution-envelope.examples.json \
  --timeout-seconds 30 \
  --json
```

## Allowlist 与输入门禁

允许：

- `adapters/*.json`；
- `drafts/runtime/**/*.envelope.json`。

拒绝：

- absolute / drive / UNC path；
- `..`、非 canonical path、project-root 越界和 root 外 symlink；
- allowlist 外 JSON、目录、特殊文件、不存在或不可读文件；
- 超过 1 MiB、非 UTF-8、duplicate key、非 JSON object；
- envelope schema/consistency 失败；
- secret scan 命中；输出只显示 rule id 和安全提示。

## 输出与 identity

scoped v2 source：

```text
project_root = project_root
relative_envelope = normalized project-relative path
envelope_content_id = sha256(raw validated envelope bytes)
scope_id = sha256(canonical(relative_envelope + envelope_content_id))
```

校验链：

- handoff `source.envelope_file` 必须匹配显式 scope；
- consumer `source_handoff_id` 必须匹配 handoff id；
- snapshot source 必须匹配 handoff source；
- snapshot id 必须匹配 handoff，并通过 canonical hash 重算；
- snapshot 返回后 envelope content id 必须保持不变。

输出不包含 absolute root、raw envelope、request `input`、payload refs、raw refs、raw stderr、descriptor 或 argv。

## 已冻结安全边界

- 只有 `snapshot-json`；无默认 representation；
- 三段 argv 全部由 reader 固定，不消费 descriptor argv；
- scoped lifecycle 为 `created → scoping → producing → validating → reading → ready → closed`；
- no retry、no write、no network、no service、no HTML、no adapter execution；
- reports 继续 request-scoped unavailable，不伪造持久 collection；
- 无 envelope 时 v1 runs/approvals/artifacts 继续 `unavailable/envelope_required`。
