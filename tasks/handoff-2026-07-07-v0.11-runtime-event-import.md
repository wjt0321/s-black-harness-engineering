## v0.11 Release 范围

`v0.11.0-runtime-event-import` 汇总以下能力包：

- `runtime event import --dry-run`：批量 event JSONL 只读预检。
- `runtime event import --commit`：向已存在 event ledger 追加连续 batch，post-check 失败按 byte size 回滚。
- consistency freeze：dry-run 输出 `plan_hash` / fingerprints，commit 可选 `--expected-plan-hash`。
- controlled write regression：event import commit 与 freeze mismatch 纳入统一回归。
